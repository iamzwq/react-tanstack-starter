# Tan React

基于 React 19 + Vite + TanStack Router 的现代 React 应用模板。

## 技术栈

- **React 19** - 最新版本的 React
- **Vite 7** - 快速的构建工具
- **TypeScript** - 类型安全
- **TanStack Router** - 文件系统路由
- **TanStack Query** - 数据获取和状态管理
- **Zustand** - 轻量级状态管理
- **Tailwind CSS 4** - 原子化 CSS 框架
- **Motion** - 动画库

## 路由

路由基于文件系统自动生成，路由文件位于 `src/routes/` 目录下。

- `src/routes/__root.tsx` - 根布局
- `src/routes/_authenticated/` - 需要认证的路由组
- `src/routes/_authenticated/index.tsx` - 首页
